# Collins landing page

Marketing automation landing page mock for **Collins** — neon dark theme, brand-aligned copy, and vector logo.

**Live repo:** [cap-jmk-launchpad/collins-landing](https://github.com/cap-jmk-launchpad/collins-landing)

## Preview locally

Open `index.html` in a browser, or serve statically:

```bash
npx serve .
```

## Structure

| Path | Purpose |
|------|---------|
| `index.html` | Landing page (11 conversion elements) |
| `styles.css` | Neon design tokens + layout |
| `script.js` | Scroll reveals, stat count-up |
| `assets/logo.svg` | Vectorized Collins mark |
| `docs/brand.md` | Brand guidelines handoff |
| `docs/design.md` | Design system tokens |
