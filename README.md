# collins-landing

Minimal static landing page for Collins, inspired by the [wearecollins.com](https://wearecollins.com) design language — pale canvas, deep onyx typography, and restrained layout.

## Develop locally

Open `index.html` in a browser, or serve the folder with any static file server:

```bash
npx --yes serve .
```

## Deploy

This repo is a plain HTML/CSS site. Host on GitHub Pages, Cloudflare Pages, or any static host by pointing the build output at the repository root.
